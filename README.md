# Crud JSF
Projeto Java Server Faces utilizando principais métodos para um CRUD.

## Tecnologias
* Web
* JSF
* MySQL
* Hibernate
* PrimeFaces

### Pré-requisitos
-------
* JDK - versão 1.8 do Java, ou mais recente;
* Netbeans - recomendamos o Netbeans IDE;
* MySQL - versão 8.0 ou mais recente;
