# Crud JSF
Projeto Java Server Faces utilizando principais métodos para um CRUD.

## Módulos
* Web
* JSF
* MySQL

Pré-requisitos
-------
* JDK - versão 1.6 do Java, ou mais recente;
* Netbeans - recomendamos o Netbeans IDE;
* MySQL - versão 5 ou mais recente;
